# vector-operation
Operations of vector

## define a vector
A vector of

$$
\vec{r} = x \hat{x} + y \hat{y} + z \hat{z}
$$

can be defined as follow

```python
r = Vect3(x, y, z)
```

and as example following code

```python
r = Vect3(1, 5, -4)
print(r)
```

will define $\vec{r} = \hat{x} + 5 \hat{y} - 4 \hat{z}$ and print

```
(1, 5, -4)
```

as output.
