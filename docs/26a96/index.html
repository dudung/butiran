table vs card for daily note

  • Sparisoma Viridi
24 Jan 2026 · 3 mins read · edit

There are two different way in butiran to archive information about daily activities. The first is using table and the second is using card, where for the later each activity has its own card, while for the former each activity is only as row in the table.

table

For the table the code is as follows.

{{< style class="table-0" >}}
id | activity | status | link | beg | and | doc
:-: | :- | :-: | :-: | :-: | :-: | :-:
01 | off wecker                      | 1 | -                        | 0600 | e0601 |
02 | hab breakfast with the cat      | 1 | -                        | 0615 | e0630 |
03 | pre to work                     | 1 | -                        | 0640 | e0700 |
04 | hab trip to work                | 1 | -                        | 0700 | e0800 |
05 | upd book energy                 | 1 | [26a46](/butiran/26a46/) | 0800 | e1200 | ✓
06 | hab lunch                       | 1 | -                        | 1200 | e1300 |
07 | con 002 zettel inc js hugo post | 1 | [26a93](/butiran/26a93/) | 1300 | e1400 | ✓
08 | fin 009 shortcode new loader    | 1 | [26a95](/butiran/26a95/) | 1400 | e1530 | ✓
09 | pre to home                     | 1 | -                        | 1530 | e1600 |
10 | hab trip to home                | 1 | -                        | 1600 | e1700 |
{{< /style >}}
