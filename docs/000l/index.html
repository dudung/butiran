svgpath

December 21, 2023

intro

  • It is a Hugo schortcode
  • It draw SVG path.
  • Required parameter is id that must be unique, width and height.
  • Current version has only one color according to colorscheme, but not responsive yet.

  width="300"
  height="200"
  border="none" >}}

l 0 100
l 100 0

{{< /svgpath >}}
