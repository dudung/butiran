inspiration

Menarik untuk mempelajari Nyquist plot atau Nyquist diagram, suatu plot respons frekuensi yang digunakan dalam teknik kendali dan pemrosesan sinyal, di mana frekuensi disapu sebagai parameter sehingga menghasilkan suatu plot berbasiskan freqkuensi [ 5 ]. Suatu plot Nyquist menyediakan suatu cara langsung dalam menentukan apakah suatu penguat akan berosilasi, yang merupakan pemanfaatkan jenis plot ini untuk analisis stabilitas [ 6 ]. Plot ini dapat digambarkan dengan cara termudah menggunakan plot Bode [ 7 ]. Wolfram Alpha telah menyediakan contoh untuk membuat plot Nyquist suatu fungsi transfer [ 8 ].
