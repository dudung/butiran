Gua Susun Utara

siasat lawan siasat

pelupa atau penipu atau lainnya

siasat lumbung palsu

darah naga

rahasia masa kecil

rencana kecil lo-kai

perkumpulan para lumbung

siluman wanita rambut emas

pengangkatan saudara

menyebarkan benih

sebuah ramalan gendeng

tugas yang gagal

penghuni baru lorong utara
