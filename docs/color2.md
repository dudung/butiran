# color2
Store color for two-dimensional shapes


## instantiation
Syntax

`Color2([stroke[, [fill]])`

+ `stroke`: stroke color, e.g. `#00f`, default `#000`.
+ `fill` : fill color, e.g. `#eef`, default `#fff`.

Code

```python
c2 = Color2('#00f', '#eef')
print(c2)
```

Output

```json
{ "stroke": "#00f", "fill": "#eef" }
```
