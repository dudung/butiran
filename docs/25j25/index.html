notes

  • It has been a while since facing the problem with inconsistency CSS appearances between local with hugo server -D and deployed page on GitHub Pages.
  • A long and exhausted discussion with misleading prompts and answers has finally provided the solution 1.

solution

  • Use --minify flag consistently in local or in hugo.yaml while generating the site remotely for GitHub Pages deployment.
  • It is use on the GitHub Action but not locally.
  • For serving with hugo
    hugo server -D --minify
    • Open http://localhost:1313/butiran/
  • For building the site locally
    hugo -d public/butiran --baseURL "/butiran" --minify
py -m http.server -d public
    • Open http://localhost:8000/butiran/
  • Remote website
    • https://dudung.github.io/butiran/
