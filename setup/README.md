# Local Setup Scripts
This folder contains optional helper scripts for working with this Hugo site on your local machine.

At the moment it includes:

| Subfolder | Purpose |
|-----------|---------|
| [`npprel/`](npprel/README.md) | Installs the custom `npprel://` protocol handler so you can click links to open Hugo posts directly in Notepad++. |


## How to Use
1. After cloning the repository, open this `setup` folder.
2. Go into the subfolder you’re interested in (currently only `npprel`).
3. Follow the instructions in that subfolder’s `README.md` (usually run an `*-install.bat` once).

New setup tools will be added here in their own subfolders as needed.
